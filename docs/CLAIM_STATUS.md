# Claim status

This repository does **not** claim a proof of the Riemann Hypothesis.

The main research note proves a structural reduction from a dyadic reciprocal Nyman--Beurling refinement to the integer Beurling / Nyman--Baez-Duarte approximation distance.

The remaining assertion that this integer distance tends to zero is the RH-level Nyman--Beurling/Baez-Duarte endpoint. It is not proved in this repository.
