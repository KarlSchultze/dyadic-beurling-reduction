# Posting guide

Suggested first post: GitHub release.

Suggested title:

**A Dyadic-to-Integer Beurling Reduction for a Reciprocal Nyman--Beurling Refinement**

Suggested framing:

- Do not say this proves RH.
- Say it is a preliminary research note and a conditional structural reduction.
- Invite mathematical review.
- Point reviewers to `docs/CLAIM_STATUS.md` before the PDF.

Possible no-invitation platforms:

- GitHub repository or release.
- Personal website or blog.
- Zenodo, if a DOI is desired.
- Math-focused discussion forums that allow public posts, following each community's rules.
