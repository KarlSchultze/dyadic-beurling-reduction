# Public graph boundary

These figures are intended for the public GitHub Pages site only.

They include:

- a visual summary of the public projection dichotomy;
- a plot of reciprocal Beurling atoms in the endpoint form
  \[
  F_{1/n}(1/x)=\rho(1/(nx))-n^{-1}\rho(1/x);
  \]
- finite-block numerical plots for the earlier reversal-across-the-decimal model
  \[
  G_K(s)=\sum_{k=1}^{K}\sum_{10^{k-1}\le n<10^k}\left(\frac{\operatorname{rev}_k(n)}{10^k n}\right)^s.
  \]

They do **not** include material from the later proof-by-contradiction research route.

The finite-block plots for \(G_K(s)\) should not be described as a proof of analytic continuation for the infinite series. Each \(G_K\) is an entire finite sum, so the figures visualize finite analytic approximants across the complex plane. The infinite-series question remains separate unless supported by a standalone written argument.
